My contributions

Combat System

When planning for this project we wanted to have a modular weapons system where you can easily create more weapons for the game just by creating a data asset. Using this data asset the component will generate the weapons based on given values. The blueprint shown here handles the hit registration of the attack.

Health System

When you want combat you want a health system, for this I made 1 component that works for everything that needs health. The object itself can handle what it wants to do based on the 'OnDamageTaken' and 'OnDeath' events. The blueprint shown here shows the TakeDamage function.
